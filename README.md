<h1 align="center"> 🔶🔸 Country App🔸🔶 </h1>

## Description

## 🌍 Country Details
- Get comprehensive details about any country, including its history, culture, and notable landmarks.
- View high-quality images that showcase the beauty and uniqueness of each country.

## 👍 Like Feature
- Express your appreciation for the countries you love by liking their profiles.
- See which countries are most popular among users based on the number of likes.

## 📜 Detailed Country Pages
- Access in-depth information about each country, including demographics, major cities, and points of interest.
- Learn about the country’s economy, government, and key historical events.
